## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Design Philosophy

The new design maintains the adventurous, authentic spirit of Super Duper Coffee while presenting it in a modern, professional way. The color palette reflects the coffee theme, and the layout emphasizes the key messages:
- Adventure-ready products
- Sustainability commitment
- Quality and freshness
- Convenience and portability

All improvements maintain the original brand voice and messaging while significantly enhancing the user experience and visual appeal.
